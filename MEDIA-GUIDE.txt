================================================================================
OFF-THE-RECORD: HOW TO ADD MEDIA POP-UPS
================================================================================

This guide explains how to add clickable emoji icons in your chapters that
open photos, videos, audio, newspaper clippings, or external links.

================================================================================
OVERVIEW
================================================================================

There are TWO types of media:

1. FILES (photos, videos, audio) - Must be uploaded to assets/ folder
2. LINKS (external URLs) - No file needed, just the URL

================================================================================
FILE NAMING CONVENTION
================================================================================

Format: ch##-short-description.ext

Examples:
  ch08-dombrowski-extension.jpg
  ch12-verlander-tale-of-tape.jpg
  ch19-verlander-trade-audio.mp3
  ch24-fantasy-camp-story.jpg

This keeps files organized and sortable by chapter.

================================================================================
STEP-BY-STEP: ADDING A PHOTO/VIDEO/AUDIO
================================================================================

STEP 1: PREPARE YOUR FILE
-------------------------
- Compress images to under 200KB using https://squoosh.app
  Settings: MozJPEG, Quality 70-75, Resize width 1200px max
- Name the file: ch##-short-description.jpg
- Save to: /assets/ folder in your project
- Push to GitHub or copy locally

STEP 2: ADD ENTRY TO data/media.js
----------------------------------
Open data/media.js and add your entry inside MEDIA_CONTENT = { }

Example for a photo:

    'my-photo-id': {
        type: 'photo',
        emoji: 'üì∑',
        label: 'Photo',
        caption: 'Description that appears below the image',
        src: './assets/ch12-my-photo.jpg',
        placeholder: false
    },

Example for a video:

    'my-video-id': {
        type: 'video',
        emoji: 'üé¨',
        label: 'Video',
        caption: 'Description of the video',
        src: './assets/ch15-my-video.mp4',
        placeholder: false
    },

Example for audio:

    'my-audio-id': {
        type: 'audio',
        emoji: 'üé§',
        label: 'Audio Recording',
        caption: 'Description of the audio',
        src: './assets/ch19-interview.mp3',
        placeholder: false
    },

STEP 3: ADD EMOJI TRIGGER TO data/chapters.js
---------------------------------------------
Find the sentence where you want the emoji to appear.
Add this code right after the sentence (before </p>):

    <span class="media-emoji" data-media-id="my-photo-id">üì∑</span>

IMPORTANT: The data-media-id MUST match the ID you used in media.js

Full example in context:

    BEFORE:
    <p>The phone rings. It's Al Avila with big news.</p>

    AFTER:
    <p>The phone rings. It's Al Avila with big news. <span class="media-emoji" data-media-id="al-avila-call">üìû</span></p>

STEP 4: BUMP CACHE VERSION IN sw.js
-----------------------------------
Open sw.js and change all three version numbers:

    const CACHE_VERSION = 'v151';
    const STATIC_CACHE = 'off-the-record-static-v151';
    const CONTENT_CACHE = 'off-the-record-content-v151';

Increment the number each time you make changes (v150 -> v151 -> v152...)

STEP 5: TEST
------------
- Clear browser cache (DevTools > Application > Clear site data)
- Or: DevTools > Application > Service Workers > Update on reload
- Refresh the page
- Navigate to your chapter
- Tap the emoji - your media should pop up!

================================================================================
STEP-BY-STEP: ADDING AN EXTERNAL LINK
================================================================================

Links don't need a file - just the URL.

STEP 1: ADD ENTRY TO data/media.js
----------------------------------
Add your entry inside MEDIA_CONTENT = { }

Example:

    'player-stats': {
        type: 'link',
        emoji: 'üîó',
        label: 'Baseball Reference',
        caption: 'Miguel Cabrera Career Statistics',
        url: 'https://www.baseball-reference.com/players/c/cabremi01.shtml',
        placeholder: false
    },

Note: Use 'url' instead of 'src' for links!

STEP 2: ADD EMOJI TRIGGER TO data/chapters.js
---------------------------------------------
Same as photos - add after your sentence:

    <span class="media-emoji" data-media-id="player-stats">üîó</span>

STEP 3: BUMP CACHE VERSION
--------------------------
Same as above - increment version in sw.js

STEP 4: TEST
------------
Tap the link emoji - it should open the URL in a new tab.

================================================================================
TROUBLESHOOTING
================================================================================

EMOJI NOT CLICKABLE:
- Check that data-media-id matches exactly (case-sensitive)
- Make sure the ID exists in media.js

IMAGE NOT LOADING:
- Check the file path in src is correct
- Make sure file is in assets/ folder
- Check for typos in filename

CHANGES NOT SHOWING:
- Clear browser cache
- Check that you bumped the version in sw.js
- Try incognito/private window
